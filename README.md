Pokédex Go
==========

A localizable pokédex for Niantic's game Pokémon Go. Currently available in English, French and German.

## Installation

    npm install --save pokedexgo

## Usage

    import Pokedex from 'pokedexgo';
    const English_Pokedex = Pokedex('EN');
